# 🍕 Pizzaria Mega Sabores

> Landing page de uma pizzaria fictícia, com foco em interface, organização de layout e uso de componentes do Bootstrap.

---

## 🚀 Preview

[Acessar projeto online](https://mega-sabores.vercel.app/)

---

## 💡 Contexto do projeto

Este projeto foi desenvolvido como entrega de curso, com total liberdade criativa, sendo construído como uma aplicação autoral.

O objetivo foi simular uma página real de apresentação comercial, aplicando conceitos de estruturação, responsividade e utilização de componentes prontos através do Bootstrap.

## 📚 Conceitos aplicados

* Estruturação semântica com HTML
* Estilização com CSS
* Utilização do Bootstrap para layout e componentes
* Responsividade para diferentes dispositivos
* Navegação com Scroll Spy
* Uso de componentes interativos (carousel e tabs)
* Organização de conteúdo em seções

---

## ⚙️ Funcionalidades

* 📸 Carousel de imagens
* 🍕 Cardápio segmentado por categorias
* 📱 Layout responsivo
* 📬 Formulário de contato
* 🔗 Navegação entre seções
* 📍 Informações e redes sociais

---

## 🎯 Foco técnico

Este projeto teve como principal objetivo o uso prático do Bootstrap para construção de interfaces, explorando componentes prontos e sua integração com HTML e CSS.
